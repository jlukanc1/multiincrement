# multiincrement
A incrementing converter in Python for WASP-OS
 * Not very fast, not tested on device.
 Add the multicount.py to your apps, add the icon from images.py to your images.py

# Shows (in order)
* Binary
* Octagonal
* Decimal
* Hexadecimal
